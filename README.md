# IBM Data Engineering Specialization

This repository contains course materials from the Coursera **IBM Data Engineering Specialization**.

---

**Master the most up-to-date practical skills and knowledge data engineers use in their daily roles:**

- **Relational Databases & DBA Concepts:**  
  Learn to create, design, and manage relational databases. Apply database administration (DBA) concepts to RDBMSs such as MySQL, PostgreSQL, and IBM Db2.

- **NoSQL & Big Data:**  
  Develop working knowledge of NoSQL and Big Data using MongoDB, Cassandra, Cloudant, Hadoop, Apache Spark, Spark SQL, Spark ML, and Spark Streaming.

- **ETL, Data Pipelines & Warehousing:**  
  Implement ETL and Data Pipelines with Bash, Airflow, and Kafka. Architect, populate, and deploy Data Warehouses. Create BI reports and interactive dashboards.
